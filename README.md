# Genesee

AI for biomedical and pharmaceutical research — by Genesee Sciences.

A terminal user interface (TUI) application built with [Textual](https://textual.textualize.io/).

## Requirements

- Python 3.10+

## Installation

```bash
# Clone the repository
git clone <repository-url>
cd genesee-cli

# Install with uv
uv sync

# Or install with pip
pip install -e .
```

## Usage

```bash
genesee
```

### Keyboard Shortcuts

| Key | Action           |
|-----|------------------|
| `q` | Quit             |
| `f` | Toggle file tree |
| `?` | Show help        |

## Development

```bash
# Install dependencies
uv sync

# Run the app
uv run genesee
```
